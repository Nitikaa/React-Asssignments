● Difference between a Library and Framework?
--> Framework is a full flegded packages. It provides the flow of the application to the developer.
Developer needs to write code as per instruction given by framework

E.g - VueJs, Angular, nodeJs

Library - independent code of unit, requested by developers.It took minumum efforts to put into our code.
the control remains with developers, who decides when to call library.
E.g - ReactJS,Jquery etc

● What is crossorigin in script tag?
-> It provides support for CROS (Cross origin resource sharing) on script tag. It allows resources sharing between differnt origins and resets the 
default behaviour of the browsers (same origin policy SOP)

● What is diference between React and ReactDOM
- React -> holds all the sources of components, state , props
- ReactDOM -> it helps React on interacting with DOM 

Why Seperate -New introduction of react Native which only works for mobile apps.
